# 商品销售数据分析报告

## 1. 项目背景
本项目旨在通过对某电商平台的商品销售数据进行深度分析，洞察不同商品大类和小类的销售趋势，识别畅销品与滞销品，为库存管理、营销策略制定提供数据支持。

## 2. 数据来源与说明
- **数据来源**：模拟电商订单数据 (假设文件名为 `sales_data.csv`)。
- **数据字段**：包含订单ID、订单日期、商品大类、商品小类、销售金额、销售数量等关键字段。
- **数据规模**：涵盖全年12个月的销售记录。

## 3. 分析方法与流程
1.  **数据清洗**：处理缺失值、异常值，转换日期格式。
2.  **特征工程**：提取“月份”、“季度”、“年份”等时间维度特征。
3.  **探索性分析 (EDA)**：
    - 分析各商品大类的月度销售趋势（柱状图/折线图）。
    - 对比不同商品小类的全年销售贡献度。
4.  **可视化呈现**：使用 Matplotlib 和 Seaborn 绘制直观图表。

- **业务建议**：建议增加热销品库存，针对滞销品开展促销活动。
